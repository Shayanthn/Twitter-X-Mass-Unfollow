# Contributing to X Unfollower 2025

Thanks for your interest! This script is actively maintained.

## How to contribute

1. Fork the repo
2. Test on the latest X.com UI
3. Make your changes (especially selector updates)
4. Open a Pull Request with clear description

## Current priorities

- Keep selectors updated with X UI changes
- Add support for mobile browser (if possible)
- Add Persian/Farsi translation in comments (optional)
